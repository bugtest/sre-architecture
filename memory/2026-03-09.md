### OpenClaw Windows 安装 - node-llama-cpp 报错解决方案

**问题：** Windows 下 `npm install -g openclaw` 时 `node-llama-cpp` 编译失败

**推荐方案（跳过 node-llama-cpp）：**
```bash
# 设置环境变量跳过编译
setx NODE_LLAMA_CPP_DISABLE_BINARY_INSTALL 1

# 重新安装（跳过脚本）
npm install -g openclaw@latest --ignore-scripts
# 或 pnpm
pnpm add -g openclaw@latest --ignore-scripts
```

**适用场景：** 不需要本地 LLM 嵌入功能时

**完整方案（需要本地 LLM）：** 安装 Visual Studio 2022 Build Tools + Desktop development with C++
