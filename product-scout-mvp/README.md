## 一、产品定位

### 一句话描述
**帮跨境电商卖家在 3 分钟内完成竞品分析，用 AI 驱动选品决策**

### 目标用户
| 用户类型 | 痛点 | 付费意愿 |
|---------|------|---------|
| Amazon 卖家 | 选品耗时、数据分散 | 高 ($50-200/月) |
| Shopify 卖家 | 找不到蓝海品类 | 中高 ($30-100/月) |
| 跨境电商新手 | 不懂市场分析方法 | 中 ($20-50/月) |
| 一件代发卖家 | 需要快速验证产品 | 高 ($30-80/月) |

### 核心价值主张
- ⚡ **3 分钟** 完成传统需要 3 小时的竞品调研
- 🎯 **AI 推荐** 蓝海品类，带数据支撑
- 📊 **一站式** 聚合 Amazon/1688/Shopify 数据
- 💰 **ROI 计算器** 自动算利润空间

## 二、MVP 核心功能（4 周版本）

### 🎯 必须有的功能（P0）

| 功能 | 描述 | 技术实现 | 优先级 |
|------|------|---------|-------|
| **1. 一键竞品分析** | 在 Amazon/1688 产品页点击插件，自动抓取竞品数据 | content script + web_fetch | P0 |
| **2. AI 选品评分** | 基于销量、评论、价格、竞争度给出 0-100 分 | OpenClaw Agent + 评分模型 | P0 |
| **3. 利润计算器** | 输入采购价，自动算 FBA 费、佣金、广告费、净利润 | 内置计算逻辑 | P0 |
| **4. 关键词洞察** | 显示该品类核心搜索词、搜索量、竞争度 | Brave Search API | P0 |
| **5. 历史趋势** | 显示 90 天价格/销量趋势图 | 本地存储 + 简单图表 | P0 |

### 📋 可以延后的功能（P1）

| 功能 | 描述 | 优先级 |
|------|------|-------|
| 多产品对比 | 同时对比 5 个竞品 | P1 (v0.2) |
| 供应商推荐 | 自动推荐 1688 供应商 | P1 (v0.2) |
| 品类监控 | 设置价格/库存提醒 | P1 (v0.2) |
| 数据导出 | 导出 Excel/PDF 报告 | P1 (v0.2) |

### 🚫 不做（至少 MVP 阶段）

- 多账号管理
- 团队协作文档
- API 开放
- 移动端 App

## 四、OpenClaw Agent 设计

### Agent 角色定义

```markdown
# ProductScout Agent

## 使命
分析电商产品数据，输出选品建议和利润评估

## 输入
- 产品 URL (Amazon/1688/Shopify)
- 目标市场 (US/EU/CN)
- 预期售价

## 输出 (JSON)
{
  "productName": "产品名",
  "category": "品类",
  "score": 78,  // 0-100 选品评分
  "metrics": {
    "estimatedMonthlySales": 1500,
    "competitionLevel": "MEDIUM",
    "priceRange": {"min": 15, "max": 35},
    "reviewCount": 342,
    "averageRating": 4.3
  },
  "profitAnalysis": {
    "costPrice": 8,
    "sellingPrice": 29.99,
    "platformFee": 4.5,
    "fbaFee": 5.2,
    "estimatedAdCost": 6.0,
    "netProfit": 6.29,
    "margin": "21%"
  },
  "keywords": [
    {"term": "wireless earbuds", "volume": 45000, "competition": "HIGH"},
    {"term": "bluetooth headphones", "volume": 28000, "competition": "MEDIUM"}
  ],
  "recommendation": "GO",  // GO / NO-GO / NEEDS_RESEARCH
  "reasoning": "竞争中等，利润率 21%，月销量可观..."
}
```

### 调用流程

```javascript
// background/service.js
async function analyzeProduct(url, targetMarket) {
  // 1. 调用 OpenClaw Gateway
  const response = await fetch('ws://127.0.0.1:18789', {
    method: 'POST',
    body: JSON.stringify({
      action: 'invoke_agent',
      agentId: 'product-scout',
      task: `分析这个产品：${url}, 目标市场：${targetMarket}`
    })
  });
  
  // 2. 解析 Agent 返回的结构化数据
  const result = await response.json();
  
  // 3. 返回给 Popup UI 展示
  return result;
}
```
